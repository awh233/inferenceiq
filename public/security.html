Security at InferenceIQ

Your data security is our top priority. We implement industry-leading security practices to protect your sensitive inference workloads.

🔐

Infrastructure Security

All data transmitted through InferenceIQ is protected with the highest encryption standards.

  • TLS 1.3 encryption for all data in transit
  • AES-256 encryption for data at rest
  • Zero-knowledge architecture - we never store your prompts or completions
  • Automatic key rotation every 90 days
  • Dedicated VPC and network isolation per customer
  • DDoS protection and rate limiting
Zero Data Retention: InferenceIQ request data is routed in real-time and never stored on our servers. We only retain metadata necessary for billing and analytics.

Data Handling & Privacy

We take a privacy-first approach to data handling and provide full transparency about our practices.

  • No storage of prompts or completions on our servers
  • Request data is routed in real-time to the selected provider
  • Metadata retention limited to 90 days for analytics
  • Full GDPR rights including data deletion and portability
  • Transparent data processing agreements for all customers
  • Annual independent security audits
Your Control: You maintain full control over your data. We never use your inference data for model training or AI improvement.
👤

Access Control & Authentication

Enterprise-grade access controls to manage who can access your account and data.

  • Role-Based Access Control (RBAC) with fine-grained permissions
  • SSO/SAML support for enterprise customers
  • API key rotation and management
  • Comprehensive audit logs for all API calls and account changes
  • IP allowlisting for additional security
  • Two-factor authentication (2FA) support
  • Session management and timeout controls
🐛

Responsible Disclosure

We take security vulnerabilities seriously and work with the security community to keep InferenceIQ safe.

If you discover a security vulnerability, please report it to security@inferenceiq.com instead of opening a public issue. We will acknowledge receipt within 24 hours and keep you updated on our progress.

  • Prompt acknowledgment of all security reports
  • Transparent communication throughout the remediation process
  • Credit and public acknowledgment for responsible disclosure (if desired)
  • No legal action against good-faith security researchers
🏗️

Infrastructure Details

InferenceIQ is deployed on secure, scalable cloud infrastructure with redundancy and failover capabilities.

  • Multi-region deployment across US, EU, and APAC
  • Automatic failover and disaster recovery capabilities
  • Regular backup and recovery testing
  • No single point of failure in critical systems
  • Continuous monitoring and alerting for security events
  • Vulnerability scanning and penetration testing

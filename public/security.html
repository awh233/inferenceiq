Security Measures

InferenceIQ implements multiple layers of security to protect your data:

  • Authentication: Multi-factor authentication (MFA) available for all accounts
  • Authorization: Role-based access control (RBAC) with granular permissions
  • Network Security: DDoS protection, Web Application Firewall (WAF), rate limiting
  • Infrastructure: Isolated VPCs, security groups, VPN support for data residency
  • Monitoring: 24/7 security monitoring, intrusion detection, anomaly detection

Responsible Disclosure

If you discover a security vulnerability, please report it responsibly to security@inferenceiq.com instead of public disclosure. We will investigate and fix critical issues within 24 hours.

Data Residency

All data is stored in compliance with your jurisdiction's requirements. We support:

  • US data residency (default)
  • EU data residency (GDPR compliance)
  • Custom data residency for enterprise customers

Business Continuity

We maintain active-active redundancy across multiple availability zones. Automatic failover ensures 99.9% availability. Daily backups with 30-day retention.
